# **Алгоритм SHA-256**
**SHA-256** - алгоритм хеширования. Он генерирует хеш - уникальный отпечаток файла, по которому можно сравнивать его содержимое с другими файлами.

Длина хеша SHA-256 - 256 бит.

Хеширование не является шифрованием. По хешу исходный файл восстановить **невозможно**.

# **Применение**

Алгоритм применяется для сравнения строк и содержимого файлов.

Например, если вы отправили файл человеку и прикрепили его хеш, получатель должен сравнить прикрепленный хеш с хешем файла, который он получил.

Если два хеша совпали, то файл не был повреждён.

Если же обнаружены различия, файл был повреждён или подменён.
