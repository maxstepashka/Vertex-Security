# **Основы работы в Vertex Security**
# **О программе**
**Vertex Security** - программа для шифрования файлов и работы с криптографическими алгоритмами. 

В приложении представлены различные алгоритмы шифрования, которые повышают безопасность хранения и передачи файлов.

Более подробно ознакомиться с принципами работы, различиями и примерами применения алгоритмов вы сможете в документации.
С её помощью вы в доступной форме изучите основы работы с алгоритмами шифрования.

# **Интерфейс программы**
Интерфейс программы состоит из трех основных компонентов:
- **Верхняя панель** - на ней расположены кнопки перехода на главную страницу, в хранилище ключей и в документацию.
- **Боковая панель** - на ней находятся вкладки с алгоритмами. Чтобы выбрать алгоритм шифрования, нажмите на кнопку с его названием.
- **Центральное окно** - в нём находятся элементы, необходимые для использования самого алгоритма.


# **Устройство главного окна**
Для окна алгоритмов **MD5** и **SHA-256** главное окно представляет собой единственное поле с выбором файла и результатом.

Для остальных алгоритмов в главном окне расположены 3 вкладки:
- **Шифрование**
- **Дешифрование**
- **Генерация ключей** - в этом окне представлен функционал генерации ключей - последовательностей символов, с помощью которых шифруются и дешифруются файлы.

# **Хранилище ключей**
**Хранилище ключей** - один из компонентов программы. С его помощью можно сохранять ключи в зашифрованном виде.

Для запуска нажмите на иконку ключа на верхней панели. Вам будет предложено задать пароль для шифрования ключей.

При каждом последующем открытии хранилища необходимо снова вводить пароль.

Хранилище предоставляет возможность сохранять ключи и пароли в зашифрованном виде, не опасаясь их кражи.

Для лучшего понимания принципов использования приложения рекомендуется обратиться к документации.

Для доступа перейдите на главную страницу или нажмите кнопку со значком вопроса на верхней панели.
