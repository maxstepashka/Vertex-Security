# **Алгоритм RSA**
**RSA** - асимметричный алгоритм. Для шифрования он использует публичный ключ, а для расшифровки - приватный (секретный).

# **Применение**

Алгоритм используется для безопасного обмена сообщениями.

Для начала необходимо сгенерировать пару из приватного и публичного ключа.

Приватный ключ необходимо хранить у себя.

Публичным ключом необходимо поделиться с тем, от кого вы собираетесь получить файл.

Вашим публичным ключом отправитель зашифрует файл.

Вы получите зашифрованную версию и расшифруете ее своим приватным ключом.

**Важно!** Никогда не отправляйте свой **приватный** ключ кому либо. Отправляется только **публичный** ключ. Приватный ключ запрашивают **только** мошенники.

##### **Будьте бдительны!**

# **Особенности работы**
Публичный и приватный ключ всегда связаны между собой.

Приватный ключ дополнительно защищен паролем. Это повышает безопасность в случае его утечки.

**Важно!** В случае утечки приватного ключа, запрещено его дальнейшее использование. 

**Незамедлительно** сгенерируйте новую пару.

# **Как использовать в Vertex Security**

Для начала работы перейдите на вкладку **RSA** из главной страницы.

## **Генерация ключей**

- Выберите длину ключа.

- Создайте папку для сохранения ключей.

- Задайте пароль для шифрования приватного ключа.

- Запустите процесс.

По окончании процесса в выбранной папке появятся два файла: **private.pem** - приватный ключ и **public.pem** - публичный ключ.

## **Шифрование**

- Откройте вкладку **Шифрование**
- Выберите публичный ключ того человека, которому вы хотите отправить файл.
- Выберите файл, который хотите зашифровать.
- Создайте файл, куда будет записываться результат, и выберите его в соответствующем поле.
- Запустите процесс.

Отправьте результат тому человеку, чьим публичным ключом вы зашифровали файл.

## **Дешифрование**

- Откройте вкладку **Дешифрование**.
- Выберите свой приватный ключ.
- Введите пароль от своего приватного ключа.
- Выберите файл, который хотите расшифровать.
- Создайте файл, куда будет записываться результат, и выберите его в соответствующем поле.
- Запустите процесс.

После окончания процесса в файле для записи результата появится расшифровка.
