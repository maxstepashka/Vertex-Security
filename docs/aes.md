# **Алгоритм AES**
**AES** - симметричный алгоритм. Для шифрования и дешифрования он использует ключи размером в 128, 192 и 256 бит.

Алгоритм AES применяется для хранения файлов в зашифрованном виде. Только владелец ключа сможет расшифровать файл.

Для шифрования файлов данным алгоритмом может быть применён **ключ** - шестнадцатеричная последовательность символов одного из ранее указанных размеров.

Кроме того, может быть применён **пароль** - любая последовательность символов. 

С помощью алгоритма **SHA-256** пароль будет преобразован в ключ размером 256 бит. 

Подробнее об алгоритме **SHA-256** в соответствующей главе.

# **Как использовать в Vertex Security**

Для начала работы перейдите на вкладку **AES** из главной страницы.

При желании использовать **ключ**, откройте вкладку **Генерация ключей**, выберите размер и получите результат.

Если же предполагается использование **Пароля**, просто введите его в поле. 

## **Шифрование**

- Откройте вкладку **Шифрование**
- Выберите тип ключа (Ключ/Пароль).
- Введите ключ или пароль для шифрования.
- Выберите файл, который хотите зашифровать.
- Создайте файл, куда будет записываться результат, и выберите его в соответствующем поле.
- Запустите процесс.

**Важно!** Процесс не удаляет исходный файл. Если предполагается хранение зашифрованного файла, безвозвратно удалите исходный!

## **Дешифрование**
- Откройте вкладку **Дешифрование**.
- Выберите тип ключа (Ключ/Пароль).
- Введите ключ или пароль для дешифрования.
- Выберите файл, который хотите расшифровать.
- Создайте файл, куда будет записываться результат, и выберите его в соответствующем поле.
- Запустите процесс.

После окончания процесса в файле для записи результата появится расшифровка.
