# **О криптографических алгоритмах**
**Криптографический алгоритм** - алгоритм, который позволяет шифровать и расшифровывать информацию.

# **Классификация алгоритмов**
Криптографические алгоритмы делятся на 3 типа:
- **Хеш-функции** - создают уникальный отпечаток файла, по которому содержимое можно сравнивать с другими файлами и проверять его целостность.
- **Симметричные** - используют один и тот же ключ для шифровки и расшифровки.
- **Асимметричные** - Используют для шифровки публичный ключ, а для дешифровки - приватный.
