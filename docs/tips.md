# **Советы по повышению безопасности**
Следующие советы помогут вам значительно повысить уровень безопасности:
- **Не забывайте очищать буфер обмена после копирования паролей и ключей.** Шифрование хранилища не распространяется на буфер обмена операционной системы.
- **Не пренебрегайте использованием больших размеров ключей.** Безопасность данных дороже места на диске!
- **Не используйте один и тот же ключ для шифрования большого количества файлов.** Чем чаще вы используете один и тот же пароль, тем злоумышленникам будет проще его подобрать.
- **Никогда не оставляйте хранилище ключей открытым.** Нажимайте кнопку перехода назад или закрывайте приложение.
