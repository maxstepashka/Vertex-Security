# Vertex Security - программа для работы с криптографическими алгоритмами
**Vertex Security** - приложение для шифрования файлов и работы с криптографическими алгоритмами.

## Как установить
Перейдите по ссылке: https://github.com/maxstepashka/Vertex-Security/tags

Выберите версию и нажмите на неё.

На странице версии выберите файл-установщик **"vertex_security.exe"** и скачайте его.
Запустите установщик и установите **Vertex** как обычную программу для Windows 10/11.

После установки в меню **"Пуск"** появится папка **"Vertex Security"** c ярлыком, который запускает программу.

## Начало работы
На главной странице нажмите на вкладку **"Изучение основ"** - там вы найдете инструкцию по использованию программы.

После изучения инструкции рекомендуется обратиться к документации - в ней подробно описаны принципы использования криптографических алгоритмов.
